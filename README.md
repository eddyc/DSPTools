DSPTools
========

Audio DSP tools for doing audio programming on Mac OS X

Note: 32bit code is legacy, and is being slowly ported the 64bit as I need it.

##Dependancies##

HDF5 - Install through Homebrew (brew install hdf5)

###Audio IO###

-   AudioInput
-   AudioObject
-   AudioStream

###DSP###

-   FFT
-   PhaseVocoder
-   TriangleFilterBank
-   BandPassFilter
-   DTWAccelerate

###Data Structures###

-   Matrix
-   AudioMatrix
-   MatrixRingBuffer

###OpenCL###

####Classes####

-   OpenCLObject

####Data Structures####

-   MatrixCL
